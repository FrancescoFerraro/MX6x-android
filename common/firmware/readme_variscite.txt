Package	
----------
SPL.mmc		- First level bootloader build to boot from sdcard
SPL.nand	- First level bootloader build to boot from nand.

The above pacakages where built by Yocto tools and u-boot for Yocto.
We modified Android u-boot in order to use 2 level bootloader.
In case you want to build the SPL please use the Yocto V5B2 and above instructions.
